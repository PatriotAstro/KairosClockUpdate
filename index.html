KairosClock USB Flasher

This updates old KairosClock devices over USB serial, even if they do not have web update support yet.

Standard Update (Preserve Settings)

Recommended for normal updates. Keeps Wi-Fi and preferences in NVS.

Recovery Update (Full Stack)

Use only if device is boot-looping or shows flash read err, 1000.

  1. Use Chrome or Edge (desktop).
  2. Connect KairosClock by USB data cable.
  3. Click Connect and choose the serial port.
  4. Install update and wait until completion.
Uses a conservative flash speed (115200) for reliability on older Macs/USB adapters. Start with Standard Update. Use Recovery Update only when needed.
